## Inference

* clone the repo `git clone https://github.com/hknhj/image-processing-project-2.git`
* Install dependencies `pip install -r requirements.txt`
* Run `python process.py --image 'input/폴더에 넣어둔 사진'` . Script will automatically download the pretrained model.
* Outputs will be saved in `output` folder.
* `output/alpha/..` contains alpha images corresponding to each class.
* `output/cloth_seg` contains final segmentation.
